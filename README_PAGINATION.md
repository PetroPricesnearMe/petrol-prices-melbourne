## üöÄ Quick Start

### Basic Implementation

```tsx
import Pagination from '@/components/common/Pagination';

function MyPage() {
  const [currentPage, setCurrentPage] = useState(1);
  const totalPages = Math.ceil(items.length / 12);

  return (
    <>
      <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4 gap-6 auto-rows-fr">
        {currentItems.map(item => (
          <div key={item.id} className="h-full">
            <Card {...item} />
          </div>
        ))}
      </div>

      <Pagination
        currentPage={currentPage}
        totalPages={totalPages}
        onPageChange={setCurrentPage}
      />
    </>
  );
}
```

### Using PaginatedGrid

```tsx
import { PaginatedGrid } from '@/components/common/PaginatedGrid';

function MyPage({ items }) {
  return (
    <PaginatedGrid
      items={items}
      renderItem={(item) => <Card {...item} />}
      itemsPerPage={12}
      gap="md"
    />
  );
}
```

## üéØ Grid Implementation

### Responsive Breakpoints

```css

sm:grid-cols-2       /* Tablet: 2 columns (‚â• 640px) */
lg:grid-cols-3       /* Desktop: 3 columns (‚â• 1024px) */
xl:grid-cols-4       /* Large: 4 columns (‚â• 1280px) */
```

### Gap Spacing

```css
gap-6                /* 1.5rem (24px) - Recommended */
```

### Uniform Heights

```tsx
<div className="grid ... auto-rows-fr">      {/* Equal row heights */}
  <div className="h-full flex flex-col">     {/* Fill cell height */}
    <div className="flex-1">                 {/* Flexible content */}
      {/* Card content */}
    </div>
  </div>
</div>
```
